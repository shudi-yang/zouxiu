项目的运行环境：
	项目需在本地服务器下运行
	所以需要搭建后台（phpnow phpstudy wamp）
	这里选择phpnow集成环境为例的安装步骤
		1.下载压缩包	到任意盘符根目录		
		2.右键解压到当前位置	
		3.右键package，解压到当前文件夹（20）	
		4.打开 开始菜单-》搜索-》cmd-》右键命令提示符，选择管理员方式打开
		5.找到刚才新建的文件夹路径		
		6.输入Setup.cmd，执行该文件	
		7.根据提示输入信息：20，回车		
		8.根据提示输入信息：50，回车		
		9.根据提示输入信息：a，回车	
		10.根据提示输入信息：y，回车	
		11.可能会有安全警告，选择允许	
		12.提示安装成功，自动打开浏览器，显示默认页面

项目的使用方式：
	把项目文件夹xiu放入本地服务器htdocs中
	浏览器地址栏输入localhost/xiu/index打开


项目说明
	此项目是一个电商网站，名为走秀网
	网站页面包括：
		首页
		登陆注册页
		商品列表
		商品详情
		购物车页面
	网站首页导航栏 可跳转到商品列表页 /
	点击不同商品可跳转到同一详情页  /
	商品详情页可加入购物车 /
	商品详情页可跳转到购物车页面
	
项目亮点
	点击不同商品能够跳转到同一个详情页，根据数据渲染详情页
	商品列表页面实现了-无限加载-功能
